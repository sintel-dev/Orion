orion.db.OrionDBExplorer.add_datarun¶

OrionDBExplorer.add_datarun(experiment, pipeline)[source]¶

Add a new Datarun object to the database.

The Datarun needs to be associated to an Experiment and a Pipeline.

Parameters

  • experiment (Experiment or ObjectID or str) – Experiment object (or the corresponding ObjectID, or its string representation) to which this Datarun belongs.

  • pipeline (Pipeline or ObjectID or str) – Pipeline object (or the corresponding ObjectID, or its string representation) used by this Datarun.

Returns

Datarun
